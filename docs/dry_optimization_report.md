## 5. 重构示例

### 5.1 订单操作处理

**优化前**:
```javascript
// 重复的switch语句处理订单操作
switch (action) {
  case 'cancel':
    this.cancelOrder(orderId);
    break;
  case 'pay':
    this.goToPay(orderId);
    break;
  // 更多case...
}
```

**优化后**:
```javascript
// 使用公共模块处理订单操作
handleOrderAction({
  action,
  orderId,
  onCancel: (id) => this.cancelOrder(id),
  onPay: (id) => this.goToPay(id),
  // 更多回调...
});
```

### 5.2 页面加载和提示

**优化前**:
```javascript
// 重复的加载和提示代码
wx.showLoading({ title: '加载中...' });
try {
  // 业务逻辑
  wx.hideLoading();
  wx.showToast({ title: '操作成功' });
} catch (error) {
  wx.hideLoading();
  wx.showToast({ title: '操作失败', icon: 'none' });
}
```

**优化后**:
```javascript
// 使用公共模块的加载和提示函数
showLoading('加载中...');
try {
  // 业务逻辑
  hideLoading();
  showSuccess('操作成功');
} catch (error) {
  hideLoading();
  showError('操作失败');
}
```
